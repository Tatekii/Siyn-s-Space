## `keyof`
访问类型上的公共属性key
## `&`
- 获取两个类型中的并集
- 如果在两个类型中有同key但类型不同的属性，该key类型为never

## `extends`

- `tuple`
	- 元祖类型扩展范型
		```typescript
		type ExtendTuple<T extends unknown[]> = [string,...T,number]

		type e1 = ExtendTuple<boolean>
		// [string,boolean,number]
		type e2 = ExtendTuple<[number,string]>
		// [string,number,string,number]
		```
