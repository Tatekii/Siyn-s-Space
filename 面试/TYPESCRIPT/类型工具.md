- `Partial`
```javascript
type Paritial<T> = {
	[P in keyof T] ?: T[P]
}
```
- `PartialOption`
```javascript
type PartialOption<T,K extends keyof T> = {
	[P in K] ?: T[P]
}
```
- `Readonly`
```typescript
type Readonly<T> = {
	readonly [P in keyof T]:T[P]
}
```
- `Pick`
```javascript
type Pick<T,K extends keyof T> = {
	[P in K] : T[P]
}
```
- `Record`
```typescript
type Record<K extends keyof any,V> = {
	[C in K] : V
}
// 与Pick,Partial,Readonly不同，R e c
```
